FalseClass

false のクラス。false は FalseClass クラスの唯一のイ ンスタンスです。false は nil オブジェクトとともに偽を表し、 その他の全てのオブジェクトは真です。

スーパークラス:

  • Object

メソッド:

self & other

常に false を返します。

self | other
self ^ other

other が真なら true を偽なら false を返します。

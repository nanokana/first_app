io/nonblock

IO#nonblock?

self がノンブロックモードならば真を、ブロックモードなら偽を返す。

IO#nonblock = bool

bool が真なら self をノンブロックモードに、偽ならブロックモー ドにする。

IO#nonblock(bool=true) { ... }

ブロック実行中、一時的に self のブロックモードを変更する。 bool が真ならノンブロックモード、偽ならブロックモードになる。

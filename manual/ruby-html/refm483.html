マニュアルダウンロード

このリファレンスマニュアルは以下からまとめてダウンロードできます。

HTML 形式 (2005-11-29版)

  • EUC tar.bz2[外部], zip[外部]
  • UTF8 tar.bz2[外部], zip[外部]

上記以外の形式などについては以下のページを参照してください。

  • HTML版 http://www.ruby-lang.org/ja/man/archive/[外部]
  • Windows HTML Help版と分割HTML http://elbereth-hp.hp.infoseek.co.jp/ruby.html[外部]
  • Rubyドキュメント http://www.ruby-lang.org/ja/20020107.html[外部]

また、最新のRD形式のソースは以下からダウンロードできます。 毎時17分頃のチェックで変更があった時に更新されます。

  • <URL:http://www.ruby-lang.org/ja/man/man-rd-ja.tar.gz>[外部] (reject 以外)
  • <URL:http://www.ruby-lang.org/ja/man/man-rd-ja-reject.tar.gz>[外部] (reject のページのみ)

頭に### rejectがついているページはman-rd-ja.tar.gz[外部]に含まれないので、 RWikiで使う場合には man-rd-ja-reject.tar.gz[外部] を展開してman-rd-ja-reject/*.rdをman-rd-ja/直下に追加すると便利です。

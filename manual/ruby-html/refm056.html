Float

浮動小数点数のクラス。Float の実装は C 言語の double で、その精度は環 境に依存します。*1

# あるシステムでの 1/3(=0.333...) の結果
printf("%.50f\n", 1.0/3)
=> 0.33333333333333331482961625624739099293947219848633

finite?

数値が ∞ でも、NaN でもなければ真を返します

infinite?

数値が +∞ のとき 1、-∞のとき -1 を返します。それ以外は nil を返 します。浮動小数点数の 0 による除算は ∞ です。

inf = 1.0/0
p inf
p inf.infinite?

=> Infinity
   1

inf = -1.0/0
p inf
p inf.infinite?

=> -Infinity
   -1
nan?

数値が NaN(Not a number)のとき真を返します。浮動小数点数 0 の 0 に よる除算は NaN です。

nan = 0.0/0.0
p nan
p nan.nan?

=> NaN
   true

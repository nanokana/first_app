io/wait

IO#ready?

ブロックせずに読み込み可能なら真を返す。そうでなければ nil を返す。 self が EOF に達していれば false を返す。

IO#wait([timeout = nil])

self が読み込み可能になるまでブロックし、読み込み可能になったら self を返す。

timeout を指定した場合は、指定秒数経過するまでブロックし、タ イムアウトした場合は nil を返す。

self が EOF に達していれば false を返す。

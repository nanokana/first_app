gem コマンドの使用例

% gem install hogehoge

パッケージ hogehoge の最新版をインストールする。

% gem cleanup hogehoge

パッケージ hogehoge の最新版以外をアンインストールする。

% gem --help [hogehoge]

その他詳しい使い方を調べる。

% gem help install
% gem_server

インストール済みのライブラリのドキュメントを http://localhost:8808/ （デフォルト）で見ることが出来ます。

インストールしたライブラリを使用するには？

例：gem でインストールした hogehoge を使用する場合

require 'rubygems'
require_gem 'hogehoge'
# 必要であればさらに hogehoge 内のライブラリを require する。

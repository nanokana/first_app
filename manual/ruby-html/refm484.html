HTML Help 版及び分割 HTML について

内容

　HTML Help 版は、Rubyリファレンスマニュアル(日本語版)と Ruby Documentation Project(日本語版)をベースに Ruby のソースなどからもドキュメントを取り入れて、 HTML Help 化したオフラインドキュメントです。

　分割 HTMLは、HTML Help 化する上で副次的に作成された HTML ドキュメントですが、 Windows 以外の OS でもオフラインドキュメントとして使用できます。

入手先

  • <URL:http://elbereth-hp.hp.infoseek.co.jp/ruby.html>[外部]

    HTML Help 版及び分割 HTML の最新版はここから入手できます。

オリジナルドキュメント(RD他)入手先

  • Ruby リファレンスマニュアル　ダウンロード[外部]

    リファレンスマニュアル(RDノード別)、RDPマニュアル(RDノード別)が入手できます。

  • Ruby Home Page[外部]

    Ruby のソースが入手できます。

オンラインマニュアル

  • Ruby リファレンスマニュアル(日本語版)[外部]

    Ruby リファレンスマニュアルがオンラインで参照できます。

  • Ruby Documentation Project(日本語版)[外部]

    RDP(Ruby Documentation Project) ドキュメントがオンラインで参照できます。
